Remembering Morocco Calendar UI architecture: Mobile view utilizes .compact-calendar component injected dynamically into .month-header loops. Event discovery agent (event_agent.py) implemented via instaloader scraping public Instagram hashtags (e.g. #gnaoua) to populate events_gallery.json dynamically. Compact calendar utilizes CSS variables for city colors (Tangier, Chefchaouen, Fez, Rabat, Essaouira, Marrakech).
