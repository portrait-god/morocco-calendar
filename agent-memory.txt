Remembering Morocco Calendar UI architecture: Mobile view utilizes .compact-calendar component injected dynamically into .month-header loops. Event discovery agent (event_agent.py) implemented via instaloader scraping public Instagram hashtags (e.g. #gnaoua) to populate events_gallery.json dynamically. Compact calendar utilizes CSS variables for city colors (Tangier, Chefchaouen, Fez, Rabat, Essaouira, Casablanca, Taghazout, Marrakech).
- Added Casablanca (14 days) and Taghazout (7 days) into the core `generate_calendar.py` routing logic.
- Implemented iOS-native fluid `touchstart`/`touchend` swipe-back listeners on popup UI tabs (Locations, Events) to return to main calendar view.
- Removed legacy visual `.timeline` progress bar from HTML for cleaner interface.
- Shifted event tracking from dynamic scraping to a statically curated `events_db` mapping array linked specifically to date indexes within `generate_calendar.py`.
